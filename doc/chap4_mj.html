4.1 Creating polyhedron

4.1-1 PolyhedronByInequalities
‣ PolyhedronByInequalities( arg )( operation )

Returns: a Polyhedron Object

The function takes a list of lists \([L_1, L_2, ...]\) where each \(L_j\) represents an inequality and returns the polyhedron defined by them. For example the \(j\)'th entry \(L_j = [c_j,a_{j1},a_{j2},...,a_{jn}]\) corresponds to the inequality \(c_j+\sum_{i=1}^n a_{ji}x_i \geq 0\).
